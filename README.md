# Alpha Ω Dragon Nuclear Geometry Lab v1.5

A browser-based interactive nuclear geometry sandbox built in pure HTML/CSS/JavaScript.

This project explores symbolic and educational visualizations of:

* proton / neutron organization
* electron shell occupancy
* excitation states
* isotope imbalance
* symbolic alpha / beta / gamma-style events
* resonance and field behavior
* stability vs instability transitions

The goal is NOT to create a literal Standard Model solver or real nuclear physics engine.

Instead, this is an exploratory visual systems lab designed to help users:

* interact with emergent structures,
* experiment with parameter spaces,
* observe stability transitions,
* and learn through direct manipulation.
